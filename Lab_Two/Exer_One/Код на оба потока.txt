1. Код для основной диаграммы последовательности действий: Основной поток
@startuml
actor Клиент
actor Турагент
participant Система

Клиент -> Система: Поиск тура
Система -> Клиент: Отображение списка туров
Клиент -> Система: Выбор тура
Система -> Клиент: Информация о туре
Клиент -> Система: Запрос на бронирование
Система -> Турагент: Запрос подтверждения бронирования
Турагент -> Система: Подтверждение бронирования
Система -> Клиент: Подтверждение бронирования
Клиент -> Система: Оплата тура
Система -> Турагент: Отправка документов клиенту
Турагент -> Клиент: Передача документов

@enduml


2. Код для вспомогательной диаграммы последовательности действий: Вспомогательный поток
@startuml
actor Клиент
actor Туроператор
participant Система

Клиент -> Система: Оставить пожелания
Система -> Туроператор: Передача пожеланий
Туроператор -> Система: Формирование тура
Туроператор -> Система: Добавление тура в каталог
Система -> Клиент: Обновление каталога
Туроператор -> Система: Запрос прогнозирования продаж
Система -> Туроператор: Отчет о прогнозировании продаж

@enduml
